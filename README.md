# 🌸loja Amonelu Stickers fofis ✨

> Uma linda lojinha virtual de stickers com design clean, papel de parede estampado e checkout PIX integrado.

Este repositório contém a aplicação da loja virtual inspirada na interface oficial da Amonelu Stickers.

## 🛠️ O que foi implementado:
* 🌸 **Papel de parede estampado:** Estampa romântica e delicada gerada nativamente por CSS.
* 🛍️ **Efeito de Stickers Autêntico:** Cards circulares com efeito de linha de corte tracejada de adesivos reais.
* 💳 **Checkout Expresso via PIX:** Integração instantânea de carrinho com popup de pagamento por Chave Copia e Cola.
